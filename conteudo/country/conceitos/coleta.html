#Conceitos[Coleta de informações]

$Significado

O próprio nome já é bastante intuitivo,a coleta de informações se trata da parte de encontrar um alvo e em seguida coletar o máximo de dados possível para realizar uma operação, seja ela qual for.

Antes de coletar as informações de um alvo, é importante lembrar que nessa parte o alvo sempre será um dispositivo, ou seja, um computador. Sendo assim, os dados que devem ser coletados são referentes a segurança, tipo de sistema e componentes que o dispositivo possui.

$Meios

Para realizar a coleta de informações de um alvo, o mais comum, após definir quem é o alvo, é usar as ferramentas apropriadas. Mas quando falamos de ferramentas, existem várias para diversos proprositos, então a melhor forma de escolher a ferramenta certa para uma tarefa seria analizando as funcionalidades que tal ferramenta possui. Esse trabalho pode ser bastante desgastante para quem ainda está iniciando na área de T.I por completo. Por isso existem sistema operacionais, baseados em linux, dotados de ferramentas apropriadas para as mais diversas finalidades. Alguns desses são: Kali Linux, ParrotOS e Black Arch Linux.

Os sistemas focados em segurança, normalmente são baseados em duas distribuições específicas do linux, o Linux Debian e o Arch Linux. Isso ocorre porque ambas as distros são relativamente simples e rudimentares. Para fins de praticidade, recomendamos que seja feita a instalação do sistema operacional Kali Linux, porque é bem mais comum e já possui diversas ferramentas pre-instaladas. Existe um tutorial padrão de distribuições linux a partir de um Windows na aba de tutoriais.

$Ferramentas

Uma vez tendo seu sistema instalado e configurado, é hora de entender melhor as ferramentas para coleta de informações. A ferramenta mais comum para isso é o Nmap, que serve para coletar diversos tipos de dados de um dispositivo, como endereço IP, quais portas estão abertas, versão do sistema, enfim, possui diversas funcionalidades.

nmap youtube.com
Saida:

As informações relevantes que o programa exibiu o endereço IP do domínio "youtube.com" e também as portas abertas que o servidor do domínio possui atualmente. O programa também pode ser usado na propria máquina para analizar as portas abertas e demais informações.
