Nessa aba estão localizadas as ferramentas mensionadas ao longo do conteúdo do site (lembrando que estamos falando do ambiente linux).

$Nmap

O Nmap (Network Mapper) é uma ferramenta de código aberto usada para descoberta e auditoria de segurança em redes.


Instalação sudo apt update
sudo apt install nmap
sudo nmap --help //para ver as opções de comandos existentes

$Gobuster

O Gobuster é uma ferramenta poderosa e muito utilizada por profissionais de segurança da informação para realizar varreduras em sites e descobrir diretórios e arquivos ocultos.

sudo apt update
sudo apt install gobuster
gobuster --help //para ver as opções de comando existentes

$Aircrack-Ng

O Aircrack-ng é uma suite de ferramentas poderosas para análise de redes sem fio. Ele permite realizar testes de segurança em redes Wi-Fi, como a captura e craqueamento de senhas WEP e WPA.


Instalação sudo apt update
sudo apt install aircrack-ng
aircrack-ng --help //para ver as opções de comando existentes

$WiFite

WiFite é uma ferramenta de linha de comando automatizada, open-source e multiplataforma, projetada para auxiliar em testes de penetração de redes Wi-Fi.

Instalação sudo apt update
sudo apt install wifite
wifite --help //para ver as opções de comando existentes

$Metasploit Framework

O Metasploit Framework é uma poderosa ferramenta de código aberto usada principalmente para testes de penetração e exploração de vulnerabilidades.

Instalação
sudo apt update
sudo apt install -y curl gpg
curl https://raw.githubusercontent.com/rapid7/metasploit-omnibus/master/config/templates/metasploit-framework-wrappers/msfupdate.erb > msfinstall
chmod +x msfinstall
sudo ./msfinstall
msfconsole //para iniciar a aplicação

$John The Ripper

John the Ripper (ou apenas John) é uma ferramenta de cracking de senhas, amplamente utilizada para testes de segurança e recuperação de senhas.

Instalação sudo apt update
sudo apt install john
john --help //para ver as opções de comando existentes

$Hydra

O Hydra é uma ferramenta de ataque de força bruta rápida e eficiente, usada para testar credenciais de login em diversos serviços de rede, como SSH, FTP, HTTP, entre outros.

Instalação sudo apt update
sudo apt install hydra
hydra -h //para ver as opções de comando existentes

$SET

O Social-Engineer Toolkit (SET) é uma ferramenta de código aberto usada para testes de penetração e simulação de ataques de engenharia social.

Instalação sudo apt update && sudo apt upgrade
sudo apt install git python3 python3-pip
git clone https://github.com/trustedsec/social-engineer-toolkit.git
cd social-engineer-toolkit
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
sudo python3 setoolkit

$Nikto

Nikto é uma ferramenta de scanner de vulnerabilidades para servidores web. É uma ferramenta de código aberto que ajuda na identificação de problemas de segurança em servidores HTTP.

Instalação sudo apt install perl libnet-ssl-perl libio-socket-ssl-perl
git clone https://github.com/sullo/nikto.git
cd nikto/program
perl nikto.pl -h URL_do_alvo
